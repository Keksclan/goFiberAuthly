# Security Policy

## About This Repository

This is an **example / test project** for
[goAuthly](https://github.com/Keksclan/goAuthly). It is not intended to be
deployed as a standalone production service.

## Reporting Vulnerabilities

> **Do not** open a public issue to report a security vulnerability, and
> **never** include secrets, tokens, or credentials in issue reports.

If you discover a security issue, please use
[GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)
to report it privately via the repository's **Security** tab â†’ **Report a
vulnerability**.

We will acknowledge your report and work on a fix as quickly as possible.
Thank you for helping keep this project safe.
